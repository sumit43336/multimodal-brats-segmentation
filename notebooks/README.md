# Notebooks - Your Workspace

## Start Here!

All your work happens in these notebooks:

### 1. Data.ipynb
- Explore the BraTS dataset
- Visualize images
- Understand the data
- Test preprocessing

### 2. Model.ipynb
- Build and test models
- Try different architectures
- Debug model issues

### 3. Training.ipynb
- Train your models
- Monitor training
- Evaluate results
- Save checkpoints

---

## How to Use

1. Start with `Data.ipynb` to understand the dataset
2. Move to `Model.ipynb` to build your model
3. Use `Training.ipynb` to train and evaluate

Keep notebooks messy - they're for experimenting!

When code works well, you can clean it up and move to `src/` folder.
